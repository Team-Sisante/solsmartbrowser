# SolSmartBrowser

**A modern, full-featured web browser designed for Samsung Smart TVs and Smart Monitors**

Turn your Samsung Smart Monitor M7 (or any TV with a web browser) into a sleek, remote-friendly internet browser – no installation, no sideloading, no developer mode hacks required.

---

## 🚀 What is SolSmartBrowser?

SolSmartBrowser is a **single HTML file** that acts as a full web browser. It runs directly in the built-in TV web browser and provides:

- ✅ Clean, dark-themed interface optimized for 1920×1080 screens
- ✅ Address bar with URL entry and search
- ✅ Navigation buttons (Back, Forward, Refresh)
- ✅ Quick-launch bookmarks for your favorite sites (Google, YouTube, Netflix, etc.)
- ✅ Full iframe-based browsing with iframe-friendly sites
- ✅ Remote control & keyboard support

Because it's a hosted web page, you can use it instantly on any device – no deployment struggles, no certificates, no USB drives.

## 🔧 Quick Setup (3 steps)

1. **Get the browser code** – See the [Setup Guide](./Docs/Setup.md) for the complete `index.html`.
2. **Deploy to Netlify Drop** – Drag & drop the `index.html` onto [app.netlify.com/drop](https://app.netlify.com/drop) (no captchas, just an email).
3. **Open on your TV** – Browse to the provided Netlify URL and bookmark it for one‑click access.

That’s it. No PC server running, no developer mode, no certification issues. The app lives on the web and loads instantly.

## 💡 Why a Web‑Based Browser App?

- Your Samsung Smart Monitor M7 does **not** support sideloaded Tizen apps or remote browser launching.
- This approach works on **any device** with a web browser – not just your current TV.
- No ongoing maintenance; just host the file once and it’s available forever.
- You can easily customise the bookmarks, colours, or even add new features (voice search, tabs, etc.).
